PROVIDEDH Collaborative Platform

Your are using the PROVIDEDH Collaborative Platform.
The platform provides the tools for working with TEI-formated XML documents, allowing for 1) Collaboratory entity and uncertainty annotation, 2) Collaborative corpus exploration, and 3) Machine-learning assisted entity unification accross the corpus.

  • PROJECTS aggregate sets of documents within a shared configuration that includes:
    • Entity names, icons and color.
    • The names (taxonomy) and color used to categorize uncertainty.
  • DASHBOARDS provide visual tools for exploring the corpus. Different charts can be mixed and matched depending on the necessity, ranging from bar charts, to sunburst visualizations and maps.
  • The DISAMBIGUATOR provides a machine-learning based analysis of the corpus for finding possible entity duplicates that can be unified.

Using TEI documents allow for a data-rich corpus by means of entity annotation, which includes adding properties to add to the completeness of the information. This process can be done collaboratively using the tools that the platform has.
